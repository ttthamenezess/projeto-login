# 🔐 Projeto Login Responsivo

Tela de login responsiva desenvolvida com **HTML5** e **CSS3**, criada como prática do capítulo 26 do **Curso em Vídeo**.

O projeto tem como objetivo praticar responsividade, estruturação de páginas web e estilização moderna para formulários de login.

---

## 🌐 Acesse o Projeto

🔗 **Deploy:** https://ttthamenezess.github.io/projeto-login/

🔗 **Repositório:** https://github.com/ttthamenezess/projeto-login

---

## 📌 Visão Geral

O **Projeto Login Responsivo** é uma página web de login adaptável para diferentes tamanhos de tela.

A interface foi desenvolvida com foco em responsividade, permitindo uma boa visualização tanto em dispositivos móveis quanto em telas maiores.

---

## ⚙️ Funcionalidades

- 🔐 Tela de login
- 📧 Campo para e-mail
- 🔑 Campo para senha
- 📱 Layout responsivo
- 🎨 Estilização com CSS
- 🖼️ Uso de imagens no layout
- 🌐 Publicação via GitHub Pages

## ▶️ Como Executar o Projeto Localmente

### 1. Clone o repositório

```bash
git clone https://github.com/ttthamenezess/projeto-login.git
```

### 2. Acesse a pasta do projeto

```bash
cd projeto-login
```

### 3. Abra o arquivo principal

Abra o arquivo abaixo no navegador:

```text
index.html
```

Também é possível executar pelo VS Code usando a extensão **Live Server**.

## 🎯 Objetivo do Projeto

O objetivo deste projeto foi praticar a criação de uma página de login responsiva utilizando apenas HTML e CSS.

Com este projeto, foram praticados conceitos como:

- Estruturação semântica com HTML
- Estilização de formulários
- Responsividade com CSS
- Organização de arquivos por pastas
- Publicação de projeto estático com GitHub Pages
- Versionamento com Git e GitHub
